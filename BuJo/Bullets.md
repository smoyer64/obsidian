The following "checkbox" combinations are available via the core Obsidian features and the minimal theme:

- [x] `- [ ]` Todo (Empty box) ✅ 2026-04-23
- [x] `- [x]` Completed (Checkbox)
- [>] `- [>]` Migrated forward (Right arrow)
- [<] `- [<]`Scheduled in the future (Left arrow)
- [/] `- [/]` In progress (Half filled)
- [-] `- [-]`Cancelled (Strikethough / Dash)
- [!] `- [!]` Important / priority (Exclamation point)
- [?] `- [?]` Follow-up (Question mark)
- [*] `- [*]` Star / Idea
- [i] `- [i]` Information
- [o] `- [o]` Event
- [p] `- [p]` Pro
- [c] `- [c]` Con
- ["] `- ["]` Inspiration
- [b] `- [b]` Book/Read
- [I] `- [I]` Idea
- [S] `- [S]` Savings/Money
- [u] `- [u]` Up / Boost / Increase
- [d] `- [d]` Down / Decrease
- [w] `- [w]` Win / Success
- [f] `- [f]` Fire / Hot
- [l] `- [l]` - Location
- `-` Note

- ! Priority
- " Inspiration
- ? Exploration
